# Klinik Pergigian Dental Heritage

A modern, high-converting dental clinic website built with React, Tailwind CSS, and Motion.

## Features

- **Premium Design**: Professional aesthetic with Teal and Wood accents.
- **Conversion Optimized**: Integrated WhatsApp booking and appointment forms.
- **Responsive**: Mobile-first design for all screen sizes.
- **Modern Stack**: React 19, Vite, Tailwind CSS 4.0, and Lucide Icons.

## Getting Started

### Prerequisites

- Node.js (v18+)
- npm

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/YOUR_USERNAME/dental-heritage-.git
   cd dental-heritage-
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Start the development server:
   ```bash
   npm run dev
   ```

## Deployment

To build the project for production:

```bash
npm run build
```

The output will be in the `dist/` directory.
